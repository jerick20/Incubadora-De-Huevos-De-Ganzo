Incubadora De Huevos De Ganzo

Control de temperatura y humedad — monitoreo y contador de días

Diseño y Conceptualización del Sistema

El sistema se basa en un microcontrolador (Arduino UNO) que recibe datos de sensores de temperatura y humedad. Estos datos se procesan y se envían a una aplicación móvil mediante un módulo de comunicación inalámbrica. La aplicación muestra en tiempo real los valores y permite el monitoreo del proceso de incubación.

Objetivos

  • Diseñar el sistema con sus componentes principales.
  • Definir la conexión entre sensores, microcontrolador y aplicación.
  • Presentar un diagrama de bloques claro y coherente.

Diagrama de Bloques del Sistema

Tipo de Comunicación Seleccionada

Se eligió Bluetooth (HC-06) por ser de bajo costo, fácil implementación y adecuado para la comunicación entre Arduino y dispositivos Android. Permite enviar lecturas en tiempo real sin necesidad de internet.

Plataforma de la Aplicación

Se seleccionó MIT App Inventor porque facilita la creación de interfaces gráficas, conexión Bluetooth y visualización de datos sin necesidad de programación avanzada. Además, permite exportar la aplicación a Android.
